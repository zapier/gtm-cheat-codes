# MCP / SDK Requirements

## MCP Use Cases

- Source discovery across Drive, CRM, Sheets, meeting notes, and content libraries.
- Read-only context gathering before drafting.
- Approval routing and message drafting.
- Tracker updates where human approval is present.

## SDK Use Cases

- Daily and weekly automations.
- Request intake.
- Approval/handoff workflows.
- Durable logging and state tracking.
- Scheduled SDR and lead-steward workflows.

## Rule Of Thumb

Use skills for judgment. Use MCP for live context and permissioned tools. Use SDK for durable recurring execution.
