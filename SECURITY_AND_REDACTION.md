# Security And Redaction

Before adding a skill, automation, example, or template, remove:

- chat workspace URLs, channel IDs, and user IDs.
- Private CRM, Coda, Databricks, Drive, or dashboard URLs.
- Personal emails and employee names unless explicitly approved.
- Customer, prospect, lead, account, and contact data.
- Private program names, unreleased roadmap details, source-system IDs, and credentials.
- Local machine paths.

## Review Checklist

- Search the changed files for private URLs, names, emails, IDs, credentials, and local paths.
- Read generated examples manually; scanners miss context.
- Confirm all examples use synthetic companies, contacts, and data.
- Keep review tools and temporary audit notes outside this repository.
