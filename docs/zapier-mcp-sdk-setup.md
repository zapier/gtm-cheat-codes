# Zapier MCP And SDK Setup

## When To Use Zapier MCP

Use Zapier MCP when a coding agent needs live context or permissioned actions across apps such as chat tools, Google Drive, Gmail, HubSpot, Sheets, Tables, or task systems.

Best for:

- Finding source context.
- Reading CRM or tracker state.
- Drafting messages from approved sources.
- Routing approval.
- Logging outcomes.

## When To Use Zapier SDK

Use Zapier SDK when the workflow should become durable, scheduled, triggered, or repeatable.

Best for:

- Daily or weekly operating summaries.
- Request intake workflows.
- Approval and handoff automations.
- Writing state to trackers.
- Posting approved outputs.

## Pattern

1. Skill defines the judgment workflow.
2. MCP gives the agent source context and safe tool access.
3. SDK turns recurring parts into deterministic automation.
4. Human review gates protect sends, posts, writes, and public publishing.
