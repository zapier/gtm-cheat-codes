# Install

## Codex

Copy this folder into your Codex skills directory and restart or reload the agent session.

## Claude Code

Copy the skill instructions into your project-level agent instructions or skill folder.

## Cursor

Use the contents of `SKILL.md` as a Cursor rule, command, or agent instruction.

## Verify

Ask the agent: "What does this skill do, what sources does it need, and what actions require approval?"
