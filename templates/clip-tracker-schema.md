# Clip Tracker Schema

| Field | Notes |
| --- | --- |
| request_id | Unique request ID |
| source_url | Original recording or source thread |
| requester | Person/team requesting clip |
| approver | Required approval owner |
| quote_or_moment | Target phrase or moment |
| status | requested, clipped, review, approved, rejected, published |
| final_asset_url | Final approved asset link |
| publish_destination | Channel or campaign |
| notes | Rights, edits, caveats |
