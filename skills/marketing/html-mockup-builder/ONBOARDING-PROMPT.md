# Onboarding Prompt: HTML Mockup Builder

Paste this into your coding agent:

```text
You are helping me configure the HTML Mockup Builder skill.

Purpose:
Builds single-file HTML mockups from campaign or page context.

First, ask me which systems I use for the required context. Then help me map my fields to the schema in SCHEMA-MAP.md. Do not send messages, update CRM records, or post publicly until I explicitly approve the draft output.

Please walk me through:
1. Required source systems.
2. Field mapping.
3. Approval and writeback rules.
4. A sanitized test run.
```
