# Setup: Customer Story Writer

## Required Sources

- Approved interview transcript, notes, or story brief
- Approved quote and metric source
- Customer profile and product context
- Usage-rights and reviewer notes

## Zapier MCP Setup

1. Create a Zapier MCP server at `https://mcp.zapier.com`.
2. Add tools for approved docs, CRM, content libraries, transcript sources, and trackers.
3. Connect each app account through Zapier.
4. Test by retrieving approved source material and drafting an internal-only story summary.

## Zapier SDK Setup

Use Zapier SDK for repeatable intake, reviewer routing, approval reminders, and story-tracker updates. Start with `npx zapier-sdk login`, then use the SDK CLI to discover the app actions and connections the workflow needs.

## Review Gates

- Customer approval
- Brand approval
- Legal approval when required
- Final publishing approval
