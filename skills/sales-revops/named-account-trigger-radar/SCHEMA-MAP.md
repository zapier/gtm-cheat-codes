# Schema Map

Map these fields to your own CRM, spreadsheet, warehouse, or Zapier Tables setup.

| Field | Required | Notes |
| --- | --- | --- |
| source_url | Yes | Canonical source record, thread, doc, meeting, or row. |
| owner | No | Workflow owner or person responsible for approval. |
| status | Yes | Draft, needs review, approved, sent, complete, or blocked. |
| notes | No | Assumptions, evidence gaps, and human review notes. |
| target_account_pool | No | Target account pool |
| crm_context | No | CRM context |
| company_signals | No | Company-approved signals |
| public_triggers | No | Public triggers |
